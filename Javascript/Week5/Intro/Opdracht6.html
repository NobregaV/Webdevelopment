1. Hoe herken je dat een waarde een String is?

Doordat je er het tussen quotes zet.

Bijvoorbeeld:

let tekst = 'dit is een string';

2. Hoe herken je dat een waarde Number is?

Doordat er een cijfer staat.

Bijvoorbeeld:

const getal = 5;

3. Hoe herken je dat een waarde BigInt is?

Als er achter het nummer een n staat dan is dat een bigint waarde

Bijvoorbeeld:

let bigint = 8n;

4. Hoe herken je de waarde Symbol?

Dit herken je doordat het Symbol wordt genoemd

Bijvoorbeeld:

let getal1 = Symbol ('g1');
