2. CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

Antwoord:

Syntax bestaat uit:

  • Selector (dat verwijst naar je html-element wat je wilt stylen)
  • Declaration (dat bestaat uit 1 of meer declarations)
  • Elke declaration bestaat uit een property naam (bij color) en een value (bijv blue)
