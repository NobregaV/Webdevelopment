1. Copy de volgende code in jouw VSCode:
let number = 3; number+=4; console.log (number);
2. Wat is de uitkomst van van de som ?

Uitkomst is 7

3. Hoe zou je het anders kunnen schrijven en dezelfde uitkomst hebben?

Geef hierbij een antwoord, geschreven code en uitleg.

Je zou het ook zo kunnen schrijven:

const sum = ++number + 3; console.log(sum);

Uitleg

Door een const variabel met de ++number waarde toe te voegen komt er automatisch 1 erbij en door +3 erachteraan te zetten komt er nog eens 3 bij.
