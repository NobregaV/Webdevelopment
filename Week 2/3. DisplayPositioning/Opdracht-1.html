1. Welke position properties zijn er? Leg voor elke position property uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

  • Static
    • Dit is de default property.

  • Absolute
    • Bij deze property komt het te staan waar de vorige element ook staat.

  • Fixed
    • Bij deze property komt het te staan bij het begin (links) van de browserpagina.

  • Relative
    • Bij deze property schuift het in dezelfde positie als de vorige positie.

  • Sticky
    • Bij deze property scrolt het mee over de website.

  • Initial
    • Bij initial gaat het terug naar de default positie.

  • Inherit
    • Bij inherit neemt hij de positie aan van de parent.
