Maak gebruik van een loop en itereer de sports array heen. Print het 1 voor 1 uit in jouw console.

Code in JS ziet er zo uit

const sports
= ["voetbal", "basketbal", "volleybal", "hockey", "honkbal", "zwemmen", "kortbal"];
for(sport of sports) {
console.log(sport);
}

Output is:

voetbal
basketbal
volleybal
hockey
honkbal
zwemmen
kortbal
