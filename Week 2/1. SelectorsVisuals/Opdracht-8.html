Cascade

Cascade houdt in dat wanneer er voor dezelde element (bijv h1) twee css codes zijn gegegeven, dan wordt de laatst toegevoegde code gehanteerd.

Deze h1 zin is groen, want de color groen is als tweede in css

Inheritance

Bij inheritance wordt de onderliggende elementen ook meegenomen in de gekozen style. Je erft iets van de parent. Alle childs krijgen dan dezelfde kleur. Bijvoorbeeld body is een inheritance, dwz dat als je body styled met color green dan wordt alles wat in de body is groen.
