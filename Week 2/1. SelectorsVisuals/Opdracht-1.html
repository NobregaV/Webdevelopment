1. Je kunt op 3 verschillende manieren CSS in je HTML inladen. Op welke 3 manieren kun je dat doen? Leg ook voor elke wijze uit wat het precies inhoud.

Antwoord

  1. External CSS: Via een externe style sheet wat je vervolgens koppelt aan je HTML-document
  2. Internal CSS: Via een style sheet in het document zelf (interne stylesheet)
  3. Inline CSS: In de html-code zelf
