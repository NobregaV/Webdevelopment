Maak een eigen voorbeeld van de volgende functies en licht toe wat de functie doet in het blok:
  1. charAt()
    2. let string1 = "dit is string 1"; let string2 = "dit is string 2"; console.log (string1.charAt());

    CharAt pakt de eerste letter (d) van de string1.

  2. concat()
    3. let string1 = "dit is string 1"; let string2 = "dit is string 2"; console.log (string1.concat(string2));

    Concat pakt de gehele eerste string1 en string2 naast elkaar

  3. slice()
    4. console.log (string2.slice(2, 5));

    Slice splits je het. In dit geval hakt hij een stuk eruit ussen letter 2 en letter 5. Dus hij laat dan alleen de letters zien tussen letter 2 en letter 5 (di)

  4. split()
    5. console.log(string1.split(" "))

    Hierbij heb ik aangegeven om alles wat een spatie heeft te splitsen.

    output is: ["dit", "is", "string", "1"]

  5. substr()
    6. console.log(string2.substr(1, 7));

    Bij Substr pakt hij de letters tussen 1 en 7 dus in dit geval de 2e letter t/m de 6e letter

    output is: n dit i
