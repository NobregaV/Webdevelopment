Er zijn verschillende selectors, bijvoorbeeld:

  • type selectors
  • descendant selectors
  • class selectors

Maak van de bovengenoemde selectors voorbeelden. Geef in je antwoord ook aan wat de voordelen zijn van elke selector.

Voorbeeld type selectors

P selecteert alleen alle elementen met <p> tag

Voorbeeld Descendant selectors

div en p zijn allebei geselecteerd om te stylen met background color geel.

Voorbeeld Class selectors

<p> heeft een class gekregen en <a> heeft ook een class gekregen met de class naam: mijnclass
.mijnclass selecteert <p class="mijnclass"> en <a class="mijnclass">

Een class selector is dus een selector van een of meerdere elementen op de pagina die een specifieke class als attribuut hebben gekregen.
